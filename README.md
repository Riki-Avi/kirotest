# ⚡ Kiro Code Lab

Plataforma de aprendizaje y ejercicios de programación en C# (.NET 10) integrada con **Monaco Editor**, compilación en tiempo real mediante **Judge0 API**, suite de **Pruebas Automatizadas (Test Cases)** y un **Tutor / Evaluador IA alimentado por Gemini 3.5**.

## 🌟 Características Principales

- 📝 **Editor Profesional (Monaco Editor)**: Editor web con resaltado de sintaxis C#, numereación de líneas y temas oscuros.
- ⚡ **Compilación Real (Judge0 API)**: Envío y ejecución del código C# directo a la API de Judge0 con consola de salida interactiva (`stdout`, `stderr`, errores de compilación).
- 🧪 **Suite de Pruebas Automatizadas (Test Cases)**: Validación de código en tiempo real contra múltiples casos de prueba estructurados (estilo LeetCode).
- 🤖 **Mentor IA Personalizado (Gemini 3.5)**: Tutor inteligente alimentado por `systemInstruction` de Gemini que evalúa tu código, da pistas pedagógicas y revisa la calidad de tus soluciones.
- 💻 **Desarrollado en C# (.NET 10)**: Backend rápido con ASP.NET Core y persistencia en JSON.

## 🚀 Instalación y Ejecución

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Riki-Avi/kirotest.git
   cd kirotest
   ```
2. Ejecutar la aplicación en .NET 10:
   ```bash
   dotnet run
   ```
3. Abrir en el navegador:
   `http://localhost:5000`
